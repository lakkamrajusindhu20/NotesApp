About the Project
A simple and user-friendly Notes Management Application built using Spring Boot and JSP.
Provides full CRUD operations (Create, Read, Update, Delete) for managing notes.
Helps users store personal notes, tasks, ideas, or reminders efficiently.
Designed with clean UI and easy navigation for quick access to notes.

Purpose of the Application
To allow users to maintain their own digital notebook.
To provide a lightweight alternative to heavy note-taking apps.
To demonstrate CRUD functionality using Spring Boot + JSP.
To help beginners understand form handling, controllers, and database operations.

Features
Create a new note
View all notes
Edit existing notes
Delete notes
